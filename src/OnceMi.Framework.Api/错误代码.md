描述框架内置功能业务异常时的错误代码。在新增功能模块时，错误代码建议不要取30001-39999中的值（作为预留）。

### 规则
1. 基本框架的错误代码范围：30001-39999
2. 原则上每个模块占用100个错误代码，如果超出100个，则占用连续的200个，依次类推
3. 命名：[模块简写]_简单描述
