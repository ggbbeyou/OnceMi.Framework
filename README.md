# OnceMi.Framework
基于.NET5/VUE开发的企业级前后端分离快速开发框架，支持国内外多种流行数据库以及统一认证授权。  

### 跑起来！
1、框架高度依赖Redis，请先准备Redis服务器。  
2、如果Redis安装在本地，直接运气即可，非本机，请修改`appsettings.json`中redis配置，然后运行本项目。
